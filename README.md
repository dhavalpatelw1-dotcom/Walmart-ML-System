# Walmart ML System

A full-stack Machine Learning application for Walmart sales prediction and analytics.

## Features

- Data preprocessing
- Machine learning model training
- Sales forecasting
- React frontend
- Python backend
- Docker deployment
- GitHub Actions CI/CD
